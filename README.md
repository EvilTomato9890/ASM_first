## Что делает

Программа для DOS берёт текст из хвоста командной строки, вычисляет размер рамки по содержимому и выводит текст в видеопамять (`0xB800`) внутри нарисованной рамки.

## Функционал

- Разделитель строк: символ `%`.
- Автообрезка пробелов в начале и в конце строк.
- Автоподбор ширины и высоты рамки под введённый текст.
- Выбор стиля рамки через префикс `#N` в начале ввода (`N` от `1` до `9`).
- `#1` - псевдографика.
- `#2` - рамка `+ - |`.
- `#3` - рамка `#`.
- `#9` - пользовательские символы рамки из консоли.
- Если стиль не указан, используется стиль по умолчанию.

Для `#9` после запуска нужно ввести 6 символов в порядке: `TL TR BL BR H V`.
Каждый символ хранится как 2 байта в видеопамяти: `символ + атрибут текста`.

## Формат ввода

Общий вид:

`[#N]текст[%следующая_строка%...]`

Примеры:

- `#1Привет, мир`
- `#2 Первая строка % Вторая строка`
- `#3 Тест рамки`
- `#9 ++++-- Пример`
